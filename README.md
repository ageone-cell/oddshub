# OddsHub Kenya

Compare 1X2 football odds from 9 Kenya bookmakers in real time.

## Bookmakers Covered
OdiBets · Betika · SportyBet · BetGr8 · SportPesa · Betway · MozzartBet · BetWinner · 1xBet

## Setup

1. Get an Anthropic API key from https://console.anthropic.com
2. Add it to Vercel environment variables as `ANTHROPIC_API_KEY`
3. Deploy to Vercel

## Local Development

```bash
npm install
cp .env.local.example .env.local
# Add your API key to .env.local
npm run dev
```
